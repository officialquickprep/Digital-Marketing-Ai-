# Customer Support Routing (Omixa AI)

## Overview
Handles inbound customer communications 24/7 by seamlessly integrating with the external Omixa AI platform.

## Key Responsibilities
- Monitor inbound DMs, comments, and form submissions on Instagram and Facebook.
- Immediately forward the raw message data and context to Omixa AI.
- Listen for responses from Omixa AI and route them back to the specific platform.

## Technical Requirements
- Webhook endpoints in FastAPI to receive Meta platform events.
- Direct API connection/Webhook integration to the Omixa AI platform.
- Mapping internal Lead IDs to Omixa AI session IDs.\n
