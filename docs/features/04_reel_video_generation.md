# Reel & Video Generation Agent

## Overview
Handles all short-form video content creation (Reels, TikToks, Shorts).

## Key Responsibilities
- Option 1 (MVP): AI writes the script, the business owner records the video, and the AI uploads it with captions.
- Option 2 (Hybrid): Business uploads raw disorganized video; AI edits it, adds subtitles and music.
- Option 3 (Full AI): AI creates full video using AI avatars and text-to-speech.

## Technical Requirements
- ElevenLabs API for Voice/Text-to-speech.
- HeyGen or Synthesia API for Avatars.
- FFmpeg or Remotion for programmatic video stitching and editing.\n
