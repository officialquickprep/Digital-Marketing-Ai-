# Content Generation Agent

## Overview
Creates all text and image-based marketing content including posts, ads, and captions.

## Key Responsibilities
- Generate platform-specific posts (Instagram vs. Facebook vs. LinkedIn).
- Generate multiple variants of Ad Copy for A/B testing.
- Produce a weekly 7-day content calendar.
- Use the RAG knowledge base + Social Media trends + Content Templates to generate highly relevant posts.

## Technical Requirements
- LangGraph Orchestrator to sequence the generation tasks.
- Image Generation APIs (DALL-E 3 / Midjourney) for visual assets.
- Approval Queue UI in Next.js where business owners can review content before it gets published.\n
