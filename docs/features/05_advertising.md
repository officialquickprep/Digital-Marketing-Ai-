# Advertising Agent

## Overview
Runs, manages, and continuously optimizes paid campaigns on Meta and Google.

## Key Responsibilities
- Configure targeting, placements, and budget limit automatically.
- Run A/B tests on creatives (generated by the Content Agent).
- Auto-pause underperforming ads and scale budgets on winners.
- Generate Lookalike audiences based on leads that converted.

## Technical Requirements
- Meta Graph API (Ads Manager) integration.
- Google Ads API integration.
- Celery Task Queue to handle long-running background optimizations (cron jobs).\n
